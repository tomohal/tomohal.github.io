Using OpenBSD as Desktop

By Tomohal Shimizu, Tue 29 December 2015, in category Os

openbsd, os

手持ちの Thinkpad X200s に OpenBSD を入れて使っているので備忘録としてセットアップについて書きます．

install packages

インストール中に xdm を使うとしておくと，xdm が起動しています(当たり前)．

csh, ksh のパッケージリポジトリ環境変数をセットします．

# vi /etc/csh.cshlogin
setenv PKG_PATH ftp://ftp.jaist.ac.jp/pub/OpenBSD/5.8/packages/amd64/"
# vi /etc/profile
export PKG_PATH=ftp://ftp.jaist.ac.jp/pub/OpenBSD/5.8/packages/amd64/"

日本語環境として必要なパッケージを入れます．

# pkg_add ja-sazanami-ttf
# pkg_add scim-anthy
# pkg_add firefox firefox-i18n-ja-39.0.3

デフォルトの window manager は fvwm です． 適当に window manager を入れます．

# pkg_add windowmaker windowmaker-extra windowmaker-lang wmclock wmmon

.xsession を編集します．

#!/bin/sh
#
PATH=/bin:/usr/bin:/usr/local/bin:/usr/X11R6/bin
WM=$(which wmaker)
export LANG=ja_JP.UTF-8
export GTK_IM_MODULE=SCIM
export XMODIFIERS="@im=SCIM"
scim -d
export LANG=C
setxkbmap -option ctrl:nocaps
xscreensaver &
if [ -x ${WM} ]; then
    exec ${WM}
else
    exec fvwm
fi

これで，Firefox に日本語入力することができます．
